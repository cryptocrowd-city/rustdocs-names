fn get_name_fee(
    op: &Operation<Self>
) -> Option<<Self::Currency as Currency<Self::AccountId>>::Balance>

Computes and returns the currency fee the sender has to pay for a certain operation. If None is returned, it means that the operation is invalid (e.g. the name is too short).

fn get_expiration(op: &Operation<Self>) -> Option<Self::BlockNumber>

For a given name operation, compute the number of blocks before the name will expire again. If None is returned, then the name will never expire.

fn deposit_fee(
    value: <Self::Currency as Currency<Self::AccountId>>::NegativeImbalance
)

"Takes ownership" of the fee paid for a name registration. This function can just do nothing to effectively burn the fee, it may deposit it to a developer account, or it may give it out to miners.
